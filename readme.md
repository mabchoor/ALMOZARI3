## Installation

1. Create a virtual environment:
    ```bash
    python -m venv myenv
    ```

2. Activate the virtual environment:
    - For Windows:
      ```bash
      myenv\Scripts\activate
      ```
    - For macOS/Linux:
      ```bash
      source myenv/bin/activate
      ```

3. Install the required packages:
    ```bash
    pip install -r requirements.txt
    ```
